# 📱 ClientFlow Pro

Application mobile Flutter pour la gestion de clients, projets et tâches.

## 🚀 Fonctionnalités

- Gestion des clients
- Suivi des projets
- Gestion des tâches
- Dashboard dynamique
- Mode démo avec reset des données

## 🛠️ Tech Stack

- Flutter
- Riverpod (state management)
- GoRouter (navigation)
- Architecture modulaire
